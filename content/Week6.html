Week6

做為工程師，不只要會找答案 ，也要會問問題。

課程內容 :

以更新 lab-booking-cdlab 倉儲內容為例，說明管理內容流程。

說明如何升級最新的 CMSimfly 倉儲中的 flaskapp.py， 並且利用 Leo Editor 的 @auto 節點指令與 @clean 節點指令，重新配置 cmsimfly.leo 開發專案檔。

Lab-Booking-CDLab 內容管理系統 :

https://mdecdlab.github.io/lab-booking-cdlab/doc/content/index.html

心得 :

遇到問題要試著解決，解決不了可以問同學或老師，也可以在 github 的 issues裡面留言( https://github.com/mdecourse/cadp2018/issues )，這週第六週了，要好好的跟上進度，遇到不懂得可以老師上傳的影片並複習之前所上的課程，也要將網站、投影片、blog 更新到第六週。
